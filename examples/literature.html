Introduction

This text will provide an introduction to the use of references with Tcl. You will need two packages from Tcllib, first the bibtex package, which again requires the cmdline package and secondly the citer package

package require bibtex
package require citer
citer::refstyle \
    -article "{author} ({year}). {title}. _{journal}_ {volume}: {pages}" \
    -incollection "{author} ({year}). {title}. In: _{booktitle}_, {publisher}. {volume}: {pages}." \
    -default "{author} ({year}). {title}. {publisher}." 
citer::bibliography literature.bib
puts [citer::getReference literature.bib Groth2013]

Groth, D., Hartmann, S., Klie, S., Selbig, J. (2013). Principal Components Analysis. In: Computational Toxicology, Humana Press. 930: 527-547.
