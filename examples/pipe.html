Python Code Evaluation

As we can use R we can as well use Python in a similar way using the pipe code chunk. Here an example:

def hello():
    print("Hello Python World!\n")

hello()
x=2
>>> def hello():
...     print("Hello Python World!\n")
... 
>>> hello()
Hello Python World!
>>> x=2

print(x)
>>> print(x)
2

Inlining Python code fragments should as well work:

The value of x is 2.
