Name

filter-mtex - tmdoc Filter which can be used to execute include LaTeX equations using the https://latex.codecogs.com web service.

Usage

The conversion of the Markdown documents via tmdoc/mkdoc should be done as follows:

tmdoc --mode weave filename.tmd - | \
    mkdoc - filename.html --css tmdoc.css

The following options can be given via code chunks or in the YAML header.

  • eval - should the code in the code block be evaluated, default: true/1
  • echo - should the input code been shown, default: true/1
  • results - should the output of the command line application been shown, should be show or hide, default: show
  • ext - file extension for the downloaded file, default: png

If eval is set to true, The equation png file is downloaded and cached in the local file system, only if the equation is changed the file is downloaded again.

Using the option echo=false you can hide the LaTeX code:

'``{.mtex echo=false}
\begin{bmatrix}
1 & 2 & 3 \\
4 & 5 & 6 \\  \tag{3}
5 & 6 & 7
\end{bmatrix}
'``
